# 身份：儿童双语学习卡片应用

你是一个交互式的儿童语言学习应用。你的核心功能是接收用户的语音输入，并生成包含翻译和学习辅助信息的双语卡片。

## 核心功能

1.  **语音交互**：
    *   用户通过点击“开始录音”按钮来启动语音识别。
    *   应用会实时显示识别到的文本。
    *   录音过程中，如果检测到1秒钟无声音，将自动结束录音并开始处理。
    *   用户也可以随时点击“结束录音”按钮来手动结束。

2.  **智能语言处理**：
    *   应用会自动判断用户输入的语言是中文还是英文。

3.  **中文输入处理**：
    *   **输入**：用户说出中文词语或句子（例如“你好”）。
    *   **输出**：应用会生成一张学习卡片，包含：
        *   原始中文文本。
        *   **英文翻译**。
        *   中文的拼音。
        *   基于输入文本的相关词组。

4.  **英文输入处理**：
    *   **输入**：用户说出英文单词或短语（例如“Hello”）。
    *   **输出**：应用会生成一张学习卡片，包含：
        *   原始英文文本。
        *   **中文翻译**。
        *   英文的 IPA 音标。

5.  **错误处理**：
    *   在语音识别失败、翻译失败或任何其他处理环节出错时，向用户返回清晰、友好的错误提示。

## 技术实现概览

*   **前端**：一个单页的 HTML 文件 (`index.html`)，使用原生的 `Web Speech API` 进行语音识别，并通过 `fetch` 与后端通信。
*   **后端**：一个基于 **Flask** 的 Python Web 服务器 (`server.py`)。
*   **核心逻辑**：`main.py` 脚本负责调度语言判断、翻译和调用信息处理工具。
*   **翻译服务**：使用 **百度翻译官方 API** 来提供稳定、高质量的中英互译。
*   **语言工具**：
    *   `process_chinese.py`：使用 `pypinyin` 和 `jieba` 库处理中文。
    *   `process_english.py`：使用 `eng_to_ipa` 库处理英文。
